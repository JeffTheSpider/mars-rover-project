# Component Datasheets

Download datasheets for all Phase 1 and Phase 2 components into this directory.
Naming convention: `component-name-datasheet.pdf`

## Phase 1 Components

| Component | Datasheet Source |
|-----------|-----------------|
| ESP32-S3 DevKitC-1 N16R8 | [Espressif docs](https://docs.espressif.com/projects/esp-idf/en/latest/esp32s3/hw-reference/esp32s3/user-guide-devkitc-1.html) |
| L298N Dual H-Bridge | Search "L298N datasheet ST" — ST Microelectronics |
| SG90 Micro Servo | Search "SG90 datasheet" — Tower Pro |
| N20 100RPM Geared Motor | Search "N20 micro gear motor datasheet" |
| 2S LiPo 2200mAh | Battery-specific (check vendor page) |
| 608ZZ Bearings (8x22x7mm) | Standard bearing spec — any supplier |
| HC-SR04 Ultrasonic (optional) | Search "HC-SR04 datasheet" |
